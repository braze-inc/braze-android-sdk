stringToMap

inline fun <T> stringToMap(string: String, entryDelimiter: String = ",", keyValueDelimiter: String = ":"): Map<String, T>

Converts a delimited string into a Map.

The string must be in the format: "key1=value1;key2=value2"

Return

A Map parsed from the string, or an empty map on error.

Parameters

string

The string to be converted to a Map.

entryDelimiter

The delimiter between entries. Default is ';'.

keyValueDelimiter

The delimiter between key and value. Default is '='.
